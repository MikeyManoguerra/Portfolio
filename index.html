LearnVenture

LearnVenture is a group project that was built with agile/scrum sprint methodology. It allows teachers to build a decision tree style teaching tool that routes the student user through checkpoints. The teacher has a lot of control over the shape of the LearnVenture, with up to four branch points, allows for multiple parent nodes, and editing. LearnVentures can be password protected and are searchable.

Views-Via-ISS

This app uses coding best practices for front end functionality and session persistence of data. The app uses three different API's, One to get the current location of the International Space Station, A second to get LandSat 8 image data of the location, and a third to get a matching static map. The LandSat 8 API returns 500 errors if no image exists per the request, and since most of the earth is water by surface area, this is a lot of errors. Views via Iss handles errors, stores successful responses, and allows changing the parameters of the location. The next step is to allow to search through your session's successful requests. Another addition would be to find another live-location data source with a circum-polar orbit.

Stack:
  • Webpack,
  • jQuery,
  • Html5,
  • Css3

Friday Night Knowledge

This project uses a spaced repetition algorithm to serve a list of foreign words in an order based upon how often you get the word correct. More successful answers mean more time before the word appears again. This is version 2.0 of the app, which is refactored from a pair project. The user can now pick from several languages, and the database will hold the history of the user for each. The code base is in a mono repo.
